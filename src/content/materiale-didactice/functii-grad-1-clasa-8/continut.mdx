## Definiție

O funcție de forma $f: \mathbb{R} \to \mathbb{R}$, definită prin $f(x) = ax + b$, unde $a \neq 0$, se numește funcție de gradul I.

Graficul acestei funcții este o **dreaptă**.

<Rezolvare>
  **Intersecția cu axele:**
  - Axa $Ox$: $f(x) = 0 \implies ax + b = 0 \implies x = -\frac{b}{a}$
  - Axa $Oy$: $x = 0 \implies f(0) = b$
</Rezolvare>
